## ✅ Project Information

![intro.png](MD_images%2Fintro.png)

- 사용자의 이력서 및 채용공고 기반 면접 질문 생성 프로그램
- 사용자의 답변에 대한 피드백 및 꼬리물기 면접 서비스 제공
- 효과적인 기업면접 대비 서비스

### 🛠️ Service Architecture

1. 사용자의 자기소개서를 임베딩

2. 자동 채용공고 크롤링 및 요약

3. 자기소개서와 채용공고의 유사도를 계산하여 면접 예상 질문 생성
